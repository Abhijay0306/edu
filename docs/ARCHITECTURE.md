# System Architecture

## Frontend
Framework: Next.js

Responsibilities:
- Student profile form
- University search
- Dashboard
- AI chat interface

## Backend
Framework: Node.js or Python

Responsibilities:
- recommendation engine
- cost estimation logic
- admission probability algorithm
- API endpoints

## Database
Database: PostgreSQL

Tables:
- countries
- universities
- programs
- admission_requirements
- scholarships
- student_profiles

## AI Layer
Uses modern LLM APIs for:
- chatbot assistance
- recommendation explanations
- visa guidance
- scholarship discovery

## External Data Sources
- university websites
- government education portals
- ranking portals

## System Modules

### 1. Profile Module
Stores student data.

### 2. Recommendation Engine
Matches student eligibility with universities.

### 3. Cost Engine
Calculates estimated study cost.

### 4. AI Advisor
Natural language guidance for students.

### 5. Scholarship Engine (future)
Search and match scholarships.

### 6. Visa Guidance Engine (future)
Provide visa information and preparation steps.
