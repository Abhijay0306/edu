# AI Study Abroad Platform – Product Requirements

## Vision
Build the world's most intelligent AI-powered platform that helps students plan their international education journey from university discovery to visa preparation.

## Core Problem
Students planning to study abroad must use multiple fragmented resources to:
- discover universities
- estimate total study costs
- determine admission eligibility
- find scholarships
- understand visa requirements

This platform centralizes these processes and provides AI-powered guidance.

## Target Users
1. Undergraduate students applying abroad
2. Professionals applying for master's programs
3. Parents planning education funding

## MVP Features

### 1. Student Profile Creation
Users input:
- GPA / Academic percentage
- Preferred country
- Course / program
- English proficiency score (IELTS/TOEFL)
- Budget

### 2. University Recommendation Engine
System recommends universities based on:
- eligibility
- budget compatibility
- admission probability
- program availability

### 3. Cost Estimation Tool
Calculate full study budget including:
- tuition fees
- living costs
- visa costs
- travel expenses

### 4. AI Advisor
AI chatbot that answers:
- admission questions
- country comparison
- scholarship advice
- visa guidance
